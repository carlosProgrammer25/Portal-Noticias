Portal-Noticias

Esse é um portal de notícias desenvolvido com Node.js, Express e MongoDB. O projeto usa Mongoose para modelagem de dados e EJS para renderização dinâmica das páginas, permitindo que as notícias sejam exibidas de forma organizada e responsiva.

No Portal-Noticias, é possível criar, editar e visualizar notícias, com todo o conteúdo armazenado no banco de dados. A estrutura do projeto foi pensada para ser simples, mas funcional, utilizando pastas separadas para páginas (pages) e arquivos públicos (public), como CSS e imagens.

Funcionalidades

Listar notícias em páginas dinâmicas

Sistema CRUD de posts (criar, editar, excluir)

Layout responsivo e organizado

Organização de arquivos em pastas separadas para melhor manutenção

Tecnologias utilizadas

Node.js

Express

MongoDB

Mongoose

EJS

HTML, CSS e JavaScript
